## Kapcsolódó SSOT

A bérlet / előfizetés költségét **nem** itt kell külön tárolni: a [[Rendszeres kiadások]] a közös Single Source of Truth. Az AYCM setup és a [[AYCM Statisztikák]] „megéri-e” kalkulációja innen olvassa az `amountHuf` + `frequency` értékeket.
