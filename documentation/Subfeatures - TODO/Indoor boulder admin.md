Business:
Ezen a felületen a beltéri falmászó termeket lehet irányítani.
CRUD műveleteket lehet velük csinálni.
Egy falmászóteremnek a következő paraméterei vannak:
- Név
	- A mászóterem neve
- Nehézségi szintek (lista)
	- Szín (pl.: kék)
		- A mászóteremben lévő út cetlijeinek színe
		- Egyedi (nem lehet ugyanolyan színből több)
	- Variáns (enum)
		- +
		- -
		- semleges
	- Alsó érték ([[Nehézségi szint skálája]] komponens)
	- Felső érték ([[Nehézségi szint skálája]] komponens)
