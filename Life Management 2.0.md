Architektúra:
- [[Backend]]: Java - Spring Boot
- [[Frontend]] + mobil (hibrid): Ionic - Angular
- [[Backend-offline first]] megközelítés
Általános gondolatok az alkalmazásról:
- Legyen minden feature feature flag-hez kötve hibrid-ben, hogy potenciálisan kiadható legyen az alkalmazás egyes feature-k nélkül
- Fel kell készülni több user használatára is majd, de elsősorban személyes használatra van az alkalmazás
- Input mezők egységesítve legyenek, és web meg mobil esetében a platformtól függjön, hogy melyiket használjuk, hogy a legkényelmesebb legyen
- [[Backend-offline first]] megközelítés!!
- Ha egy felületen egyértelmű, vagy egy input mező, amit egyértelműen használni fog a user (különben miért nyitotta volna meg), akkor legyen automatikusan aktív az input, hogy gépelhessen
